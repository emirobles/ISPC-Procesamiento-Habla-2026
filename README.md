# Portafolio de Proyectos - Técnicas de Procesamiento del Habla

## Sobre este proyecto

Este repositorio reúne las actividades, prácticas y proyectos desarrollados durante la materia **Técnicas de Procesamiento del Habla**.

Espero aprender los fundamentos del procesamiento digital de señales de audio, reconocimiento y análisis de voz, extracción de características, transformadas como Fourier, técnicas de muestreo y aplicaciones actuales relacionadas con la inteligencia artificial.

También me interesa comprender cómo se aplican estas herramientas en asistentes virtuales, transcripción automática, biometría por voz y sistemas inteligentes.

## Áreas de interés en Ciencia de Datos

- Procesamiento del Lenguaje Natural (NLP)
- Deep Learning
- Procesamiento del Habla
- Visión Artificial
- Machine Learning
- Análisis de Datos
- Inteligencia Artificial aplicada

## Objetivo personal

Desarrollar habilidades prácticas que me permitan integrar modelos de IA y análisis de datos en proyectos reales orientados a soluciones innovadoras.
